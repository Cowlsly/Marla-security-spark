# Marla Security Spark – Roadmap

A simple but useful Android app featuring a button that provides internet security tips and occasional random facts.

Built with Kotlin.

**Version: v0.4** (+0.0.3)

## Vision

A lightweight, always-available security awareness tool that makes learning cybersecurity fun and accessible.

## Current Status (2026-06-15, v0.4)

- Basic app structure exists
- Core button functionality in place
- CI workflow updated to build self-contained from checked-in source (no longer depends on zip)
- Ecosystem doc polish (spelling/grammar/punctuation) completed across Cowlsly repos
- Unified account sync work referenced (profile/avatar from central source)

## Development Phases

### Phase 1 – Core Features
- Expand tip database
- Add categories (Phishing, Passwords, Privacy, etc.)
- Random fact mode
- Simple settings

### Phase 2 – Polish & Engagement
- Better UI / animations
- Daily tip notifications (optional)
- Favorites / history
- Theming options

## Long-term Goals
- Integration with other Cowlsly tools (Account Centre for profile, web for public hub)
- Community tip submissions (future)
