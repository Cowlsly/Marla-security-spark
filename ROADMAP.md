# Marla Security Spark – Roadmap

A simple but useful Android app featuring a button that provides internet security tips and occasional random facts.

Built with Kotlin.

## Vision

A lightweight, always-available security awareness tool that makes learning cybersecurity fun and accessible.

## Current Status (June 2026)

- Basic app structure exists
- Core button functionality in place

## Development Phases

### Phase 1 – Core Features
- Expand tip database
- Add categories (Phishing, Passwords, Privacy, etc.)
- Random fact mode
- Simple settings

### Phase 2 – Polish & Engagement
- Better UI / animations
- Daily tip notifications (optional)
- Favorites / history
- Theming options

## Long-term Goals
- Integration with other Cowlsly tools
- Community tip submissions (future)
